## 2021-02-19

```shell
git clone https://github.com/honjun/hexo-theme-sakura.git
# 包装包
tyarn
# 安装hexo
tyarn global add hexo
# 本地预览
hexo server -p 4000
# 生成静态文件
hexo generator
# 远程部署
hexo deploy
# 一键部署
hexo clean && hexo d - g
```
